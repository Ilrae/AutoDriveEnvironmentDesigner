## 0. 문서 목적

이 문서는 `AutoDriveEnvironmentDesigner` 프로젝트를 아이디어 단계에서 실제 실행 가능한 졸업작품/연구 프로젝트로 옮기기 위한 상세 작업 체크리스트다.

핵심 원칙:

- 처음부터 큰 시스템을 만들지 않는다.
- 항상 "작은 성공"을 먼저 만든다.
- 현재 최우선 목표는 `CARLA + Python + OpenDRIVE 최소 성공`이다.
- 강화학습, 고급 커리큘럼, 복잡한 지도 생성은 기초 검증 이후로 미룬다.

프로젝트 한 문장 정의:

`CARLA에서 사용할 OpenDRIVE 도로를 자동 생성하고, 차량 주행 결과를 평가해서 다음 도로를 다시 생성하는 시스템`

핵심 구성 요소:

- 도로 생성기
- CARLA 실행기
- 평가 로직
- 난이도/커리큘럼 업데이트 로직
