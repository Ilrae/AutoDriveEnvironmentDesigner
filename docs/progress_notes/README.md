# Progress Notes

이 폴더는 졸업작품 진행 과정에서 나온 작업 기록, 실험 결과, 오류 수정, 마일스톤 정리를 세션 단위 Markdown 파일로 보관하는 용도입니다.

## 사용 목적

- 설치 및 환경 구축 과정 기록
- 기능 구현 단계별 결과 기록
- 오류 원인과 해결 과정 정리
- 중간 마일스톤 정리
- 최종 보고서 작성용 근거 자료 축적

## 파일 작성 규칙

- 파일명 형식: `YYYY-MM-DD_주제.md`
- 한 파일에는 한 세션 또는 한 마일스톤을 중심으로 기록
- 사용한 명령, 결과, 오류, 해결 방법, 다음 단계까지 함께 정리

## 권장 구성

1. 작업 목적
2. 환경 정보
3. 실행한 명령
4. 결과
5. 오류 및 해결
6. 결론
7. 다음 작업
